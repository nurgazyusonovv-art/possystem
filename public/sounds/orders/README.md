# Заказ үндөрү

Бул папкага заказ номерлери боюнча үн файлдарын салыңыз:

```
public/sounds/orders/1.mp3
public/sounds/orders/2.mp3
...
public/sounds/orders/100.mp3
```

- Ар бир файл ошол номердеги заказдын **даяр болгонун жарыялаган** үн болсун
  (мисалы: «Биринчи заказ даяр», «Экинчи заказ даяр» …).
- Колдоого алынган формат: `.mp3`.
- Заказ номери **100дөн ашканда кайра 1ден** башталат (циклдик).
  Мисалы: №101 → `1.mp3`, №250 → `50.mp3`, №300 → `100.mp3`.
- Эгер тиешелүү файл бул папкада **жок болсо**, баскыч басылганда
  эч кандай ката чыкпайт — үн жөн гана ойнобойт.

Үн логикасы: `src/lib/sound.ts` → `playOrderReady(orderNumber)`.
